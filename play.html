Luck Card

LUCKはこのゲームの「毒」であり「救い」である。場に出すだけで空気が変わる唯一無二の1枚。

WIN グー・チョキ・パー すべてに勝利する。
TIE LUCK vs LUCK は「あいこ」。カードはPOTへ蓄積。
呪い 第10ラウンドでLUCK同士があいこになった場合、POT内の全カードが VANISH（消滅）する。
⚠ 第10ラウンドの呪い
最後の一枚で「あいこ」が発生した場合、LUCKの持ち主であっても、POT内のすべての点数はVANISHする。

戦略： LUCKを温存するほど第10ラウンドでのリスクが高まる。POTが積み上がった終盤、LUCKを切る誘惑に抗えるか——それがSENOだ。
Variant Modes

基本ルール（OVER）に飽きた者へ。各モードは独立して遊べる。

OVER
DEFAULT
あいこ蓄積 / ジャックポット
進行各プレイヤーは手札10枚を裏向きで持つ。全員、伏せたまま1枚を選び、場の中央に裏向きで出す。「せーの！」の合図で全員同時にオープン。
勝利単独勝者がいる場合 → その人が場の全カード＋POTに蓄積されたカードすべてを獲得。
あいこあいこの場合 → 出されたカードはPOT（共有プール）に蓄積され、次のラウンドに持ち越し。これを手札がなくなるまで繰り返す。
VANISH
COLD MODE
あいこ消滅 / 冷徹モード
進行OVERと同じ。1枚を伏せて出し、「せーの！」で同時公開。
勝利単独勝者がいる場合 → その回に場に出たカードのみ獲得（POTは存在しない）。
あいこあいこの場合 → 出されたカードはすべて場から除外（捨て札）。強いカードをいつ切るか、意思決定が最も鋭くなるモード。
DARK
FATE MODE
運命モード / 戦略ゼロ
準備ゲーム開始時、自分の手札10枚をシャッフルし、裏向きの山にする。中身は確認できない。
進行毎ラウンド、山の一番上の1枚をそのまま場に出す（選択権なし）。「せーの！」で同時公開。
勝敗OVERと同じ（あいこはPOTへ蓄積）。唯一の戦略は「相手の山にあと何枚LUCKが残っているか」のカウンティング。
DUAL
CHOICE MODE
二択モード / 完全情報の読み合い
準備各プレイヤーは手札をシャッフルし裏向きのまま台にカードを設置。手札の上から2枚を選び、裏向きで自分の前に並べる（中身は全員に見える）。全員、他プレイヤーの2枚をすべて確認できる状態になる。
進行「せーの！」の合図で、自分が並べた2枚のうち1枚を選んで場に出す。残った1枚はそのまま手札に戻す。次のターンはまた手札の上から一枚を自分の前に出して2枚にする。
勝敗OVERと同じ（あいこはPOTへ蓄積）。
SHOUT
BLUFF MODE
宣言制 / 心理戦モード
進行各プレイヤーは伏せて1枚を選ぶ。場に出す前に、全員順番に「何を出すか」を宣言する（例：チョキ / LUCK 等）。宣言内容は嘘でも構わない。
公開全員の宣言が終わったら「せーの！」で同時公開。
勝敗OVERと同じ（あいこはPOTへ蓄積）。ジャンケンがポーカーになる瞬間。3〜4人推奨。
SYSTEM
MEMORY MODE
神経衰弱モード / 記憶と運
準備全プレイヤーの手札（合計40枚）をシャッフルして裏向きで場に並べる。各プレイヤーは手札を持たない。ターン制で進行。
進行全員でせーの！で1枚めくる。めくったカードでじゃんけんの勝者になっていれば、そのターンで表になったカードは自分のものに。同じ手（あいこ）なら、両方とも裏向きに戻す（位置は記憶しておく）。
LUCKLUCKを引いた場合は即座に獲得（特殊効果）。全カードが取り尽くされたら終了。記憶＋確率＋LUCKの不確定要素。
